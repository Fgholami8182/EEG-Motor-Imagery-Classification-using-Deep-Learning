# EEG Motor Imagery Classification using Deep Learning

This repository presents a deep learning framework for **Motor Imagery EEG Classification** developed as the final project for the **Special Topics in Artificial Intelligence** course.

The project investigates several convolutional neural network architectures for decoding EEG signals and compares their performance on a benchmark Brain-Computer Interface (BCI) dataset.

---

# Project Overview

Electroencephalography (EEG) records electrical activity generated by the brain. Decoding Motor Imagery EEG signals is one of the fundamental problems in Brain-Computer Interface (BCI) systems.

This project implements a complete deep learning pipeline including data preprocessing, augmentation, model training, and performance evaluation.

---

# Dataset

**BCI Competition II – Dataset III**

Dataset characteristics:

* EEG Motor Imagery signals
* Binary Classification
* 6 EEG Channels
* 896 Samples per trial
* 268 Trials
* Two classes:

  * Class 0
  * Class 1
